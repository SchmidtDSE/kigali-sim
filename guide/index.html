Montreal Policy Simulator Guide

Hello! This website provides a series of tutorials intended to help you get started with the Montreal Policy Simulator. If you are new, consider starting with the first tutorial.

Tutorials

This guide has the following tutorials each with their own bite-sized activity:
  • What is the Montreal Policy Simulator?
  • Model a single substance
  • Model a single policy
  • Expand to multiple substances
  • Build a permit system
  • Interpret results
  • You have been programming all along
  • Grow the simulation
  • Simulate with probabilities and uncertainty
  • Model energy consumption
All numbers are intentionally fake and used only for demonstrative purposes. Please keep in mind that this software is still under development. We encourage your ideas as our living software continues to grow based on your feedback. However, please note that things may also change and this guide may be refined.

Formal documentation

In addition to the tutorials, formal documentation on the engine is also available. This living specification document provides a robust technical description of this work.
